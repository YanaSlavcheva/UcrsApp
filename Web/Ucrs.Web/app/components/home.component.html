Welcome to Ucrs - University Courses Registration System

What can we do in it?

What pages do we have?
  • Login
  • Register
  • Courses (available after login) - display and managing of courses
What will you find on the Courses page?
  • Grid with all courses
  • First all courses that student is registered in are displayed, then all that the student is not registered in
  • Student can create new courses and they appear in the grid immediately
  • Student can update a course
  • Student can delete a course
  • Student can register for course only if a/ student is not already registered; b/ student has less than the max points user can get from registering in each course (100)
